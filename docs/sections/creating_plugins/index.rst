Creating plugins
================

.. rst-class:: header-text

    This session is meant to get you started with writing your first AiiDA plugin.

    If you haven't done so already, please **first** watch the `introduction to the AiiDA plugin system <https://www.youtube.com/watch?v=bjTUnHXZ6oY&list=PL19kfLn4sO_-e_A9lVYb_NBNcwoVvUP6V>`_ (~20 minutes).

    For this session, pick one of three options depending on your interests and experience:
